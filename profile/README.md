# MeanScoop

> Local discovery platform for events, businesses, deals, and community content

## What we build

- **Local Discovery** — Find events, businesses, and deals happening around you
- **Community Content** — User-generated reviews, posts, and recommendations
- **Business Tools** — Analytics, promotions, loyalty programs, and marketing campaigns
- **Content Intelligence** — AI-powered ingestion and enrichment from local sources

## Get started

- 📱 [Mobile App](https://github.com/meanscoop/meanscoop-app) — React Native (Expo) app for iOS and Android
- 🖥️ [Payload CMS](https://github.com/meanscoop/meanscoop-payload) — Backend API and content management
- ⌨️ [CLI](https://github.com/meanscoop/meanscoop-cli) — Command-line tools for data management
- 🔍 [AI Scrapers](https://github.com/meanscoop/ai-social-scrapers) — Intelligent content ingestion pipelines
- 🐳 [Stack](https://github.com/meanscoop/stack) — Docker Compose infrastructure

## Architecture

```
meanscoop-app          →  Mobile client (Expo / React Native)
meanscoop-payload      →  API + CMS (Payload 3 / Next.js)
meanscoop-cli          →  CLI tooling
ai-social-scrapers     →  Content ingestion & enrichment
stack                  →  Docker infrastructure (Postgres, Redis, etc.)
```
